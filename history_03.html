전통 경제 소유권 관념 — 보편성 재확인
교수 강조

유라시아 대륙 전체(동쪽 끝 조선 ↔ 서쪽 끝 영국), 최소 서기 100년 ~ 1800년까지 약 1,700년 동안 공통된 관념:
생산 수단에 대한 소유권은 중첩적으로 존재 → 직접 일하는 사람의 권리도 보장
자본주의의 배타적 사적 소유권 = 인류 보편이 아닌 200년도 안 된 특수한 제도

시험 포인트: "물욕 자체는 있었으나 물욕을 무한히 확장하는 것은 낯설었다." 이 문장 그대로 기억. 공동체 바운더리를 뚫는 무한 이윤 추구 = 전통 시대엔 없던 것.
01 도시 길드와 산업자본주의의 충돌
길드(Guild) — 도시판 촌락 공동체
정의

도시 경제 활동 종사자들의 동종 조합
구두쟁이는 구두쟁이끼리, 포목상은 포목상끼리, 바이올린 장인은 바이올린 장인끼리
→ 도시에서 농촌 촌락 공동체와 동일한 역할 수행

기능 내용
경제 규제 생산량·유통량의 한도 설정 → 공동체 범위 내 물욕만 허용
내면화 규제 이전에 이미 구성원 스스로 초과 생산을 상상하지 않음
소속감 삶의 정체성·지위·인간관계의 중심 (현대 게임 길드와 비슷한 소속감)
길드 내 직업 계층 구조
계층 역할 생산 수단 관계
장인 (Master) 작업장·생산도구 소유자, 작업 총괄 명목상 소유자 — 그러나 마음대로 처분 불가
직인 장인과 거의 동급의 기술 보유, 협력자 작업장·도구에 관습적 권리 보유
도제 배우는 견습생 (시다) 같은 작업장 생계 걸림
핵심

장인의 작업장·생산 도구 = 장인 + 직인 + 도제 복수의 생계가 걸린 재산
→ 농촌 지주의 토지와 동일한 논리 = 소유권 중첩
→ 장인이 팔거나 처분하려면 얘네들과 협의 필수. 처분 시 직인이 인수하거나 직인 출신 장인에게 넘김.

교수 예시 — 16~17세기 신실한 장인

선교를 위해 이스라엘로 떠나려는 장인 → 작업장·도구를 혼자 마음대로 팔 수 없음.
반드시 직인·도제와 협의 후, 직인 중 한 명이 승계하거나 그들을 받아주는 다른 장인에게 거래.

산업자본주의와 길드의 충돌 — 지오바니 스토리
교수 핵심 예시 — 밀라노 구두쟁이 지오바니

상황: 잘 나가는 장인 지오바니. 매출 100 → 120 → 140 성장. 길드 안에서 착실히 운영 중.

변화: 어느 날 산업자본주의 관념을 가진 자본가가 옆에 대형 B 구두 공장 건설.
→ 농촌에서 쫓겨 나온 노동자들을 고용 → 속성 훈련만으로 기계 조작 가능
→ 분업+기계+규모의 경제 → 압도적으로 싼 가격으로 대량 생산 개시

결과:
- 지오바니는 평범한 공산품 생산자 → 가격 경쟁에서 완패 → 수 년 내 파산
- 파산 = 작업장·도구 헐값에 처분 (이미 시장 논리가 지배 → 협의 없이 처분)
- 생산 수단에서 유리됨 → 구두 공장에 취직 시도

취직 후:
- 평생 구두 만든 장인인데 → "가죽 붙이는 일만" 배정됨
- 노동 과정(기획~완성) 전체 관할 → 단순 반복 작업자로 전락
- 쉽게 대체 가능한 존재가 됨

박탈감 포인트 — 시험 중요

농촌 출신: 경작 → 공장 노동 → 일하는 방식의 변화 (지위 변화 덜 느낌)
도시 장인: 존경받는 마스터 → 대체 가능한 단순 노동자 → 급격한 지위 하락 + 박탈감
→ 초창기 산업자본주의에 가장 격렬하게 반대한 집단 = 수공업자·장인 출신

교수 비유 — 현대판

랩을 운영하며 제자들을 이끌던 교수가 갑자기 대체 가능한 반복 업무 담당자가 됨.
수술실 메인 의사가 단순 보조로 전락. → 장인 계층이 느낀 충격이 이 정도.

02 Displaced — 삶의 유동성 폭발
Displaced의 의미와 사회 현상
Displaced (학자 표현)

자기 자리에서 오랫동안 해왔던 삶의 형태가 갑자기 날아감 → 굉장히 빠르고 급격하게
→ 삶의 유동성(mobility)이 폭발적으로 증가

현상 내용
Time Discipline 충격 자연 리듬·자율적 시간 관리 → 공장의 엄격한 출퇴근 시간 규율 → 극심한 스트레스
소리 충격 새소리·바람소리만 듣던 농촌 출신 → 공장 기계음 적응 불가. 현대에도 시골 어르신의 대로 공포감과 동일 메커니즘
베가본드(Vagabond) 유랑자·떠돌이 급증. 적응 실패자들이 돌아다님
훌리건식 폭력 집단으로 약탈·파괴. 당시 최대 사회 문제
러다이트 운동 기계 파괴 운동 (Luddite). 공장 기계를 직접 파괴
폭음 문화 목요일~금요일 폭음 → 주말 뻗음 → 월요일 몰아서 일. 도시에서도 유사한 자율적·불규칙 리듬
핵심 요약: 1,700년 이상 이어온 경제 관념이 불과 수십 년 만에 붕괴 → 그 충격이 19세기 초 서유럽의 최대 사회 문제로 부상 → 지식인들이 대책을 논의하기 시작.
지역 차이: 영국이 가장 빠름. 벨기에·프랑스·이탈리아 북부가 상대적으로 빠름. 산업화가 안 된 지역은 초창기에 그대로 유지. 그러나 충격이 워낙 커서 언론을 통해 보편적 파장 발생.
03 3대 이념 — 보수주의 · 자유주의 · 사회주의
3대 이념 비교 — 산업화(I) · 자본주의(C) 에 대한 입장

① 보수주의
Conservatism

  • I(산업화) ❌ C(자본주의) ❌
  • 둘 다 거부 → 산업자본주의 이전으로 회귀
  • 공동체·직분 강조
  • 가장 격렬하게 반응 (스펙트럼 한쪽 극단)
  • 대표: 낭만주의 시인 등

② 자유주의
Liberalism

  • I(산업화) ✅ C(자본주의) ✅
  • 사회 문제 인정하지만 IC 더 발전하면 해결됨
  • 개인의 자유 = 공동체와 무관한 개인 결정권
  • 결국 현재 우리가 사는 세계 (21세기 승자)
  • 스펙트럼 반대쪽 극단

③ 사회주의
Socialism

  • I(산업화) ✅ C(자본주의) ❌
  • 산업화는 긍정, 자본주의만 비판
  • 스펙트럼 중도
  • 자본주의 비판 = 올드한 소스(전통 경제)에서 출발
보수주의 — 더 깊이
원래 보수주의의 특징 (현재와 다름!)

보수주의는 기득권이 바뀌면 내용도 바뀌는 이념 → 가장 변화 많은 이념
이 시기 보수주의가 지키려 한 것: 산업자본주의 이전의 공동체·직분 구조
→ 지금 우리가 보는 "자본주의 친화적 보수주의"와 정반대

교수 예시 — 지주(최참판)의 입장

산업자본주의 이전: 땅 소유 + 농민들 지대 + 인간적 존경 + 안정적 수입
산업자본주의 이후: 시장 조사·경쟁·기술 개발·은행 대출·노동자 파업… 훨씬 복잡하고 존경도 없음
→ 지주는 자본가로 변신보다 과거 체제 복귀를 원함 → 보수주의 지지

보수주의 핵심 가치

공동체 + 직분(役割) = 누군가가 특정 직분을 맡으면 그 역할을 안정적으로 유지시켜야 함
→ 신분제 정당화 포함. 농민은 농사, 지주는 지도자 역할을 지속하는 사회.

자유주의 — 진짜 의미 (오해 주의!)
자유주의의 '자유' = 공동체에 대한 개인의 자유

핸드폰 던지고 훌쩍 떠나는 자유 ❌
생산 수단을 아무와 상의 없이 처분할 자유
무한한 이윤 추구를 할 자유
생산·유통·투자를 개인이 자유롭게 결정할 자유 ✅

냉전 왜곡 포인트: 냉전 시대에 자유주의 = "진정한 자유를 주는 것"으로 가르침 → 사실은 공동체 규율에서 개인(자본가)을 해방시키는 이념. "그렇게 가르치면 골치 아프니까 안 가르쳤다" (교수 직접 언급).
교수 예시 — 자유주의의 실제 내용

최참판이 농민들과 상의 없이 땅 팔 수 있어야 함 = 자유주의
길드 구성원과 상관없이 생산 왕창 늘릴 수 있어야 함 = 자유주의
→ 공동체의 바운더리와 무관하게 개인 결정에 법적 정당성 부여하는 이념

사회주의 — 자본주의 비판의 두 축
자본주의 비판 ① 유통 형식(시장 경제) 비판

시장에 buyer(구매자)와 seller(판매자)만 두면 불공정 발생
이유 1: 시장 참여자들의 체급(몸집)이 다름 → 수요공급 메커니즘 교란 가능
이유 2: 그 부의 획득 과정이 정당한지 알 수 없음 (열심히 일한 것인지, 사기인지, 상속인지 시장은 확인 불가)
→ 대안: 제3자 개입으로 시장 조절
     극단: 제3자가 생산·유통 전체 결정 = 계획 경제
     완화: 제3자가 개입해서 조절만 = 복지 국가형

자본주의 비판 ② 생산 양식(임금 노동제) 비판

생산 수단에 대한 배타적 사적 소유권 문제화
→ 생산 수단은 복수의 생계가 걸린 특별한 재산 → 개인이 독점하면 불공정
→ 대안: 생산 수단을 개인이 배타적으로 소유 못하게 개혁

자본주의 비판의 원천 = 올드한 소스

사실 이 비판은 새로운 것이 아님.
전통 경제에서 제3자 = 촌락 공동체 (농촌) / 길드 (도시) 가 이미 있었음.
소유권 중첩도 이미 있었음. → 자본주의 비판은 과거 전통 경제를 레퍼런스로 삼은 것.
(순수 논리로 자본주의를 비판한 것은 나중의 마르크스)

04 사회주의 — 드디어 완성된 정의
사회주의 정의 (최종)
✦ 사회주의 정의

19세기 초 서유럽의 사회 혼란 속에서 출현한 경제 구상으로,
산업화(I)는 긍정하되 자본주의(C)를 비판하여 보다 공정한 세상을 만들려 한 이념.

구체적 비판 내용:
① 시장 메커니즘에 대한 제3자 개입 (유통 형식 비판)
② 생산 수단에 대한 사적 소유권 문제화 (생산 양식 비판)

핵심 체크

사회주의는 역사적 개념 → 산업자본주의 없으면 사회주의 없다
사회주의 ≠ 무조건 재산 분배 / 자유 억압 / 독재 / 중화학공업 / 군사 문화
사회주의가 건드리는 것: 생산 수단에 대한 소유권 (그것도 복수의 생계가 걸린 규모 이상)

05 사회주의에 대한 오해 5가지 — 반드시 구별
냉전 시대 왜곡 vs 사회주의 원래 개념
✗
오해 ① 사회주의 = 모든 재산을 나눠 갖는 것 (예금·BMW·개인 재산 몰수)
→ 사회주의가 문제 삼는 것은 오직 생산 수단만. 일정 규모 이상, 복수의 생계가 걸린 것만. BMW는 생산 수단 아님 → 건드리지 않음.
소련에서도 개인 텃밭 소유·복권 당첨금으로 휴양지 가는 것 = 가능. 단 그 돈으로 땅·공장 사는 것 = 불가.
✗
오해 ② 사회주의 = 모든 자유 억압
→ 거부하는 자유는 딱 2가지뿐: ① 생산 수단을 개인이 배타적으로 처분하는 자유, ② 생산·유통·투자를 무한히 확장하는 자유. 다른 자유와는 관계없음.
✗
오해 ③ 사회주의 = 독재 필수
→ 사회주의 개념 자체와 독재는 무관. (마르크스의 프롤레타리아 독재는 나중 개념이며 이것과도 구분 필요.) 생산 수단 소유권 개혁 = 독재 없이도 가능.
✗
오해 ④ 사회주의 = 중화학공업 위주 / 소비재·문화 없음
→ 생산 수단 소유권 이야기와 중화학공업은 무관. 20세기 현실 사회주의 국가들에서 나타난 역사적 현상이지 사회주의 개념의 필연이 아님.
✗
오해 ⑤ 사회주의 = 군사 문화 / 고문 / 핵
→ 전혀 관련 없음. 전혀 다른 역사적 맥락에서 형성된 현상들. 사회주의 원래 개념과 무관.
교수 강조: 이 5가지 이미지는 냉전 시대에 사회주의를 반대하는 쪽과 심지어 사회주의를 지지하는 쪽 모두가 잘못 유포한 것. 사회주의자라고 스스로 부르는 사람들도 재산 평등을 사회주의라고 오해하는 경우 많음.

② 생산·분배·투자의 무한 이윤 추구를 허용하는 시장 메커니즘

06 T/F 예상 포인트 — 3차시 함정 정리

📌 길드 · 도시 충돌 관련

F
길드는 농촌 촌락 공동체와 달리 생산량을 규제하지 않았다.
→ 길드도 생산·유통량을 규제했음. 공동체 범위를 초과하는 물욕을 억제 — 촌락 공동체와 동일한 역할.
T
길드 장인의 작업장·생산 도구는 장인 소유이지만, 직인·도제도 그 재산에 관습적 권리를 가졌다.
→ 소유권 중첩. 복수의 생계가 걸린 재산 → 장인 단독 처분 불가.
F
산업자본주의에 가장 먼저, 가장 격렬하게 반대한 집단은 농촌 농민이었다.
→ 도시의 수공업자(장인 출신)가 가장 격렬하게 반대. 지위 하락과 박탈감이 더 컸기 때문. 농촌은 일하는 방식의 변화 수준.
T
장인이 공장 노동자가 되면서 노동 과정 전체 관할 → 단순 반복 작업으로 전락했다.
→ 핵심 변화. 기획~완성까지 전 과정 관할 → 가죽만 붙이는 대체 가능한 존재.

📌 3대 이념 관련

F
보수주의는 항상 자본주의와 시장 경제를 옹호한다.
→ 원래 보수주의는 산업화·자본주의 둘 다 거부. 현재의 자본주의 친화적 보수주의는 보수주의가 많이 바뀐 결과. 기득권이 바뀌면 보수주의 내용도 바뀜.
F
자유주의의 '자유'는 인간이 어떤 구속도 없이 자유롭게 사는 것을 의미한다.
→ '공동체에 대한 개인의 자유' — 생산 수단 자유 처분, 무한 이윤 추구, 생산·유통·투자의 개인 결정권. 냉전 시대에 왜곡된 이미지가 우리 통념.
T
사회주의는 3대 이념 중 정치 스펙트럼 중도에 해당한다.
→ 보수(I·C 모두 거부) ↔ 자유(I·C 모두 찬성) 사이에서 I만 찬성하고 C만 반대.
F
자유주의는 IC를 발전시키면 사회 문제가 저절로 해결된다고 보았으며, 사회 문제의 존재 자체를 부정했다.
→ 사회 문제의 존재는 인정했음. 단 IC가 충분히 발전하면 해결된다고 봄.
T
초기 사회주의의 자본주의 비판은 전통 경제(촌락 공동체·길드)의 관행을 레퍼런스로 삼은 것이다.
→ "올드한 소스". 마르크스처럼 순수 논리로 비판한 것이 아니라 과거 전통 경제에서 근거를 가져옴.

📌 사회주의 정의·오해 관련

F
사회주의는 모든 사유 재산을 부정한다.
→ 복수의 생계가 걸린 규모 이상의 생산 수단만 문제. 개인 소비재·예금·BMW 등 생산 수단이 아닌 재산은 건드리지 않음.
F
사회주의는 산업화(기계화·대량 생산)를 반대한다.
→ 산업화는 긍정. 자본주의만 비판. 사회주의 = I ✅ C ❌.
T
사회주의에서 시장 경제 비판의 핵심은 buyer와 seller에게만 맡기면 체급 차이와 부의 획득 과정의 불투명성 때문에 불공정하다는 것이다.
→ 제3자 개입 필요성의 근거. 체급 차이 + 부의 정당성 알 수 없음.
F
소련의 독재·군사 문화·중화학공업 위주는 사회주의 원래 개념에서 필연적으로 도출된 것이다.
→ 사회주의 원래 개념과 무관. 전혀 다른 역사적 맥락에서 형성된 것들.
F
사회주의가 거부하는 자유는 표현·사상·신체의 자유 등 인간의 모든 자유다.
→ 거부하는 자유는 딱 2가지: 생산 수단 배타적 처분의 자유 + 무한 이윤 추구의 자유. 다른 자유는 무관.
T
사회주의는 19세기 초 서유럽의 사회 혼란 속에서 등장한 경제 구상이다.
→ 사회주의 정의 그 자체. "경제 구상" / "19세기 초 서유럽" / "사회 혼란" 모두 포함해서 기억.

✦ 논술형 대비 핵심 논지 (3차시)

  • 도시 길드는 어떻게 농촌 촌락 공동체와 같은 역할을 했으며, 산업자본주의와 어떻게 충돌했는가?
  • 보수주의·자유주의·사회주의는 각각 산업화와 자본주의에 대해 어떤 입장을 취했는가? 세 이념을 비교하라.
  • 자유주의의 '자유'가 의미하는 것은 무엇인가? 우리가 흔히 생각하는 자유와 어떻게 다른가?
  • 사회주의 정의를 구체적으로 서술하고, 우리가 갖고 있는 오해들이 왜 사회주의 원래 개념과 무관한지 설명하라.
  • 초기 사회주의의 자본주의 비판이 왜 '올드한 소스'에 기반했다고 할 수 있는가?
