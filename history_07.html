01 루이 오귀스트 블랑키 (Louis Auguste Blanqui)
프롤레타리아(Proletariat) 개념 확정 — 15인 재판
✦ 역사적 장면 — 외워!

재판장: "당신의 직업은 무엇이요?"
블랑키: "프롤레타리아입니다."
재판장: "그게 무슨 직업입니까?"
블랑키: "직업이 아니라, 일을 해서 먹고 살지만 권리를 빼앗긴 사람들을 의미하는 것이다."

프롤레타리아(Proletariat) 정의 — 블랑키가 확정

원래 의미: 빈민계급
블랑키 이후: 임금 노동자 = 생산수단에 대한 권리가 없는 사람들
= 무산계급 (유산계급 ↔ 부르주아의 반대)
이후 노동자·프롤레타리아는 교차 사용. 특히 '프롤레타리아'는 부르주아와 대비해 자부심의 계급 의미로 사용

블랑키의 혁명 방법론 — 핵심
블랑키즘의 핵심 공식

소수의 의식화된 엘리트 → 음모적(비밀) 봉기 = 쿠데타 → 국가 권력(경찰·행정) 탈취 → 혁명적 독재로 사회주의 개혁 추진

단계내용포인트
① 주체 소수의 자각한(의식화된) 엘리트 대중봉기 ❌ — 소수 엘리트가 시점·방법을 스스로 판단
② 방법 음모적 봉기 = 사실상 쿠데타 비밀결사 조직 (사계절회 등)
③ 탈취 후 국가기구(경찰·행정) 활용해 개혁 추진 파괴가 아니라 장악·활용
④ 정치체제 혁명적 독재 (다당제 ❌) 국가기구 단독 추진 → 다당제·입법 논의 불필요
⑤ 경제 개혁 생산수단 사적 소유권 폐지·제한 + 시장 개입 단, 점진적·유연하게 (소생산자 배려)
블랑키 vs 마르크스 — 시험 포인트

블랑키: 물적 토대 성숙 여부와 무관하게 엘리트의 의지로 봉기 시점·방법 결정
마르크스: 충분한 물적 토대 성숙 + 사회 구조 변화가 전제되어야 혁명 가능
→ 블랑키 = 주체(의지) 강조 / 마르크스 = 구조(조건) 강조

독재의 의미: 이 맥락에서 독재 = 다당제를 허용하지 않는 것. 군부독재와 다름. 국가기구가 혁명 주체만으로 단독 추진 → 후에 '프롤레타리아 독재' 개념의 원형.
공산주의 용어 의미 확정 — 블랑키파의 공산주의 연회
공산주의 연회 (1840년대)

블랑키파의 데자미·필로가 주최 → 까베에게 함께하자고 요청
까베: "너희는 너무 폭력적·국가 주도적" → 참여 거부
결과: 까베 없이 블랑키파만의 공산주의 연회 → 공산주의 의미 = 블랑키즘으로 확정

확정된 공산주의 의미

이후 스스로 공산주의자라 부르는 사람들 = 사실상 블랑키주의자
핵심: 소수 엘리트 음모적 봉기 → 국가 권력 탈취 → 혁명적 독재 → 사회주의 개혁
(생산수단 소유권 철폐는 기본 전제로 깔되, 방법론이 공산주의의 정체성이 됨)

별명 & 경력

별명: "갇힌 자" — 평생 33년 감옥에서 보냄
비밀결사: 사계절회가 가장 유명
48년 혁명, 71년 파리코뮌(이때는 미리 잡혀 참여 못함) 등 수차례 봉기 시도·실패 반복

02 빌헬름 바이틀링 (Wilhelm Weitling)
바이틀링 프로필 & 사상
항목내용
출신독일 지역 수공업자 출신. 프랑스 장교의 사생아, 어머니 손에서 가난하게 성장
직업재봉사(테일러) — 유랑 직인으로 여러 도시 돌며 산업자본주의 폐해 목격
특이점지금까지 사회주의자 중 실제 노동계급 출신. 지식인이 아님
활동파리로 이동 → 의인동맹 가담. 의인동맹 내 마르크스와 논쟁 → 결별
대표 저작가난한 죄인의 복음서
말년미국 아이오와 공동체 실험(처참한 실패) → 뉴욕에서 재봉틀 기술 발명가로 변신
바이틀링의 핵심 사상

블랑키 방식(소수 엘리트 봉기·국가 탈취)에 강한 기독교적 색채 결합
명언: "예수는 최초의 공산주의자다"
사유재산 철폐 + 공동생산 → 기독교 정신으로 정당화
중앙관리 주체: 지적 엘리트 아닌 일하는 사람들(노동자) 중에서 능력자

바이틀링 vs 까베 — 시험 포인트

공통: 기독교적 성격, 국가 주도 개혁, 사유재산 철폐
차이: 까베 = 지식인·엘리트 주도 / 바이틀링 = 일하는 사람 중 능력자 주도
→ 마르크스와 논쟁: 바이틀링이 마르크스를 "먹물(지식인)"이라 공격

마르크스와의 논쟁 (의인동맹)

바이틀링 → 마르크스: "나는 직접 일하는 사람을 대표한다. 쟤는 책 읽는 먹물이다."
마르크스 → 바이틀링: "이 인간은 허황되고 종교적이고 낭만적이다."
→ 둘 다 공산주의자를 자처하지만 노선이 다름

발명가로의 변신 — 교수 언급 포인트

재봉사 출신 → 재봉틀(미싱) 기술 발전에 핵심 역할. 뉴욕에서 특허 보유.
발명가 = 분업 체계에 저항하는 존재 → 사회주의적 관점에서 연결됨
(19세기 후반 미술공예운동가들이 사회주의자 많은 것과 같은 논리)

03 루이 블랑 (Louis Blanc) — 국가주의 BUT 비(非)공산주의
루이 블랑 프로필 & 사상
항목내용
출신나폴레옹 시대 고급 외교관의 아들. 나폴레옹 몰락 후 집안도 몰락
경력법대 포기 → 변호사 사무실 서기 → 법률 지식 습득 → 언론계(잡지 발간) → 저작 노동의 조직으로 명성
대표 저작노동의 조직(L'Organisation du Travail)
48년 혁명장관 입각 → 국가작업장 실험 → 보수파 다수 당선으로 실패 → 영국 망명 (20년)
귀환파리코뮌 때 귀환 → 코뮌 방식과 맞지 않아 반대 → 쓸쓸한 말년
루이 블랑의 핵심 사상 3가지
핵심 ① 생산수단 사유재산 = 모든 악의 근원

생산수단 사유재산화 → 전면 부정
(소비재 소유권은 건드리지 않음 — 사회주의 기본 원칙)

핵심 ② 임금 균일화 — 루이 블랑의 특이점!

모든 노동은 귀천이 없다 → 임금을 균일하게
당시 생시몽주의자들(능력별 차등 임금)·프루동(숙련도별 차등)과 정면 대립
※ 임금 균일화는 사회주의 핵심 요소 아님 — 블랑 개인의 특이 주장

핵심 ③ 사회작업장 (Ateliers Sociaux) — 방법론

기존 공장 정부 인수 + 새 공장은 정부 펀드로 건설
운영은 아래로부터 노동자가 자율 관리
실현 방법: 의회 입법으로 → 폭력혁명·봉기 없음

루이 블랑 vs 공산주의(블랑키) 핵심 차이

공산주의: 소수 엘리트 봉기 → 국가 탈취 → 혁명적 독재
루이 블랑: 의회 입법 → 사회작업장 설립 → 법적·점진적 개혁
→ 같은 국가주의지만 방법론이 정반대 (폭력 ❌ vs 폭력 ✅)

루이 블랑의 현대적 의의

당시엔 비판 받음 (국가 계급성 무시, 도덕 의존 등)
But 현대 관점: 의회 입법을 통한 사회주의 = 가장 현실적인 방법론
→ 후에 개량주의·사민주의 사회주의에 큰 영향

국가작업장 실패 원인

부랑자·거지까지 몰려들어 취업 요구 → 규율 붕괴
도덕적 동기 부족 → "시간을 주면 도덕이 함양될 텐데" 기회 없이 48혁명 종료
나폴레옹 사촌이 대통령 당선 → 사회주의적 성격 탈각

04 피에르 프루동 (Pierre Proudhon) — 무정부주의자
프루동 프로필
항목내용
출신전형적인 농촌 전통경제 → 산업자본주의에 의해 해체. 10살에 인생을 알았으나 12살에 노동자가 됨
직업인쇄소 노동자 → 글 읽을 수 있게 됨 + 불온 서적 접촉 → 의식화
출판사 시도10대 후반 동료와 출판사 창업 → 바로 망함
전환점고향 장학생으로 선발(28세) → 파리대학 입학 → "대학은 사치다" 자퇴
독학3개월(39.11~40.2) 25권 독서 → 대표작 소유란 무엇인가 집필
"소유란 도둑질이다" — 사유재산권 논리적 비판
✦ 프루동의 핵심 명제 — 반드시 외워!

"소유란 도둑질이다 (Property is theft!)"
= 생산수단을 개인이 배타적으로 소유하는 것 = 도둑질

프루동이 왜 중요한가

이전 사회주의자들: 사유재산 비판 = 도덕적 비판 위주 ("꼭 그렇게 많이 가져야겠느냐")
프루동: 논리적·체계적 비판 — 사유재산을 정당화하는 근거들을 하나씩 논파
마르크스 이전 가장 큰 사회주의자로 불리는 이유

사유재산권 정당화 근거 3가지 논파
근거 ① 자연권론 — "소유는 천부적 자연권이다"

주장: 소유권 = 생존권·행복추구권과 같은 천부적 자연권 (프랑스 혁명 인권선언 기초)

프루동의 반박: 천부적 자연권이라면 보편적이어야 함
그러나 역사를 보면 사적 소유권을 주장한 것은 항상 소수뿐이었음
→ 보편성 없음 → 자연권이라 할 수 없음

근거 ② 선점의 원칙 — "먼저 갈았으니 내 것이다"

주장: 인간의 생존권을 실현하려면 땅을 갈아야 하고, 먼저 점유해서 오래 갈았으면 소유권 인정

프루동의 반박: 선점 논리의 근거는 '생존권' → 뒤에 태어난 세대도 생존권이 있음!
선점한 사람만 소유하면 나중에 태어난 사람의 생존권은?
→ 토지는 소유(영구 독점)가 아닌 용익(살아있는 동안 활용)만 가능
→ 다음 세대가 활용할 기회를 남겨야 함 → 소유는 도둑질

교수 보충 설명

법적 근거는 대부분 타임라인을 무시하는 경향. 동시에 출발했으면 선점도 이해할 수 있지만, 현실은 시대마다 다른 조건으로 태어남 → 선점 논리의 한계.

프루동 요약 정리
프루동의 위치

국가주의 사회주의자들과 달리 무정부주의 — 국가기구 자체에 반대
사유재산권을 논리적으로 비판한 최초의 체계적 작업
이후 마르크스가 이 논리를 더욱 발전시킴
3번째 비판 근거(노동가치론 관련)는 7차시에서 시간 관계상 미완 → 이후 학습 예정

→ 소수 엘리트의 음모적(비밀) 봉기 = 사실상 쿠데타. 대중봉기와 구분.

블랑키의 혁명 방법론에서 '독재'는 다당제를 허용하지 않는 것을 의미한다.
→ 군부독재와 다른 개념. 국가기구 단독 추진 = 다당제 논의 불필요.
F
블랑키는 경제 개혁도 신속하고 단호하게 추진해야 한다고 주장했다.
→ 정치(권력 탈취)는 단호하게, 경제 개혁은 점진적·유연하게. 소생산자 배려 때문.
T
공산주의 연회에서 까베가 참여를 거부하면서 공산주의의 의미가 블랑키즘으로 확정됐다.
→ 까베 빠진 블랑키파 단독 연회 → 공산주의 = 봉기·독재 방법론으로 확정.
F
블랑키는 물적 토대가 성숙해야 혁명이 가능하다고 보았다.
→ 마르크스의 입장. 블랑키는 물적 토대와 무관하게 엘리트 의지로 시점 결정.

📌 바이틀링 관련

T
바이틀링은 지금까지 학습한 사회주의자 중 실제 수공업자(노동계급) 출신이었다.
→ 재봉사 출신 유랑 직인. 나머지는 모두 지식인·변호사·귀족 계열.
T
바이틀링은 "예수는 최초의 공산주의자다"라고 주장했다.
→ 대표 명언. 기독교적 공산주의의 상징.
F
바이틀링은 까베와 달리 지식인 엘리트가 중앙관리를 해야 한다고 주장했다.
→ 정반대. 바이틀링 = 일하는 사람 중 능력자. 까베 = 지식인·엘리트 중심.

📌 루이 블랑 관련

T
루이 블랑은 의회 입법을 통한 사회작업장 설립을 주장했다 — 폭력혁명 없이.
→ 블랑키와의 핵심 차이. 같은 국가주의지만 방법론이 다름.
T
임금 균일화(모든 노동은 귀천이 없다)는 루이 블랑의 특이한 주장으로 사회주의의 핵심 요소가 아니다.
→ 정확. 생시몽주의자·프루동 등은 숙련도에 따른 임금 차등을 지지.
F
루이 블랑의 사회작업장은 국가(공무원)가 직접 운영·관리한다.
→ 아래로부터 노동자가 자율 관리. 국가는 공장을 소유·설립하되 운영은 노동자.
T
루이 블랑의 아이디어는 후에 개량주의·사민주의 사회주의에 큰 영향을 주었다.
→ 교수 직접 언급. 의회·입법을 통한 사회주의 = 가장 현실적 방법론.

📌 프루동 관련

T
프루동은 사유재산권을 도덕적 비판이 아닌 논리적·체계적으로 비판한 최초의 인물이다.
→ 마르크스 이전 가장 큰 사회주의자로 불리는 이유.
T
프루동은 선점의 원칙을 비판하면서, 토지는 소유가 아닌 용익(살아있는 동안 활용)만 가능하다고 주장했다.
→ 뒤에 태어난 세대의 생존권 논리.
F
프루동은 국가주의 사회주의자로 분류된다.
→ 무정부주의자. 국가기구 자체에 반대 — 국가주의자들과 정반대.
