01 프루동 — 사유재산권 3가지 근거 논박 완성
슬라이드 정리 — 3가지 근거 전체 구조
근거정당화 주장프루동 논박 핵심
① 법적 근거
자연권		 소유권 = 생존권·행복추구권과 같은 천부적 자연권 역사상 항상 소수만이 주장했음 → 보편성 없음 → 자연권 아님
① 법적 근거
선점의 원칙		 먼저 땅을 갈아 점유했으니 소유권 인정 생존권이 근거라면 뒤에 태어난 세대도 생존권이 있음 → 소유 아닌 용익만 가능
② 심리적 근거
(인간 본성·보편 동의)		 소유 욕구 = 인간 본성 → 모든 이가 보편적으로 동의 (로크) 평등에서 출발해 불평등으로 귀결 = 논리적 모순
계약 전제 = 평등한 주체 → 결론은 불평등 옹호
③ 경제적 근거
(사회적)		 내가 노동을 투입했으니 내 것 (개간 등) 노동에는 사회적 기여(지식·도구)가 이미 결부됨 → 혼자 한 것 아님
③ 경제적 근거
(물적)		 내 노동으로 새것을 만들었으니 소유 가능 새로 창출한 것이 아니라 주어진 자연을 변형한 것에 불과 → 소유 근거 없음
✦ 심리적 근거 논박 — 핵심 포인트 (8차시 신규)

로크의 논리: 평등한 주체들의 보편적 동의 → 소유권 정당화
프루동의 반박: 평등에서 출발해서 불평등으로 귀결 = 논리적 모순
→ 소유권이 공격받을 때 "능력 있는 사람이 더 갖는 건 당연" 식의 불평등 논리로 전환
→ 시작(평등)과 결론(불평등)이 정반대 = 모순

✦ 경제적 근거 논박 — 8차시 핵심 (가장 강력한 논박)

사회적 논박: 노동에는 기존 사회의 지식·도구가 결부됨 → 혼자 한 것이 아님
물적 논박: 자연을 변형했을 뿐, 새로운 것을 창출한 것이 아님 → 소유 근거 불성립
→ 이 경제적 근거 논박이 사회주의자들도 공감했던 가장 강력한 반박

02 프루동 — 부조주의 (Mutualism) 완전 정리
부조주의 핵심 구조
공산주의 비판 — 핵심 명제

"중앙이 모든 것을 공동체로 귀속시키는 공산주의는 노예제!"
→ 이카리아·블랑키 방식의 중앙 권력·엘리트 통치 = 직접 생산자를 노예로 만드는 것

항목내용
지향점 직접생산자의 자유 — 소규모 자영농·자영업자가 용익권으로 활용하며 자유롭게 사는 사회
소유권 생산수단 사적 소유권 불허 / 대신 용익권·점유권 인정
(살아있는 동안 활용하는 권리 — 공화주의와 차이!)
대규모 생산수단 광산·철도·대공장: 조합이 소유 (국유화 ❌)
핵심 슬로건 "관리받는 공동체원보다는 자유로운 조합원" (오언 전통 계승·발전)
경제 도구 Labor Notes + People's Bank (인민은행) — 쌈짓돈 모아 공동체 건설
정치 구조 Free Commune → 지역 Commune → Confederation (탈중앙집중화)
반대 대상 지성인/엘리트 통치·중앙 관료제 반대 → 무정부주의적 사회주의의 시초
성격 자족적·목가적 → 훗날 테러주의로 변모 (중간고사 이후 학습)
프루동 vs 공화주의 — 헷갈리는 포인트

공화주의: 소규모 소유권 인정 (자영농·장인의 소유권 있음)
프루동: 소유권 불허, 용익권만 인정 (살아있는 동안 활용만)
→ 대규모 생산수단 반대는 공통, 소규모 소유권 여부가 핵심 차이

프루동 vs 오언 — 계승과 발전

오언 전통 계승: 아래로부터 생산자 중심, Labor Note 유사 도구
오언과의 차이: 오언 = 투자자(큰손)가 공동체 건설, 관리받는 구성원
프루동 = 인민은행(쌈짓돈) + 자유로운 조합원 — 국가·큰손 없음

무정부주의와 히피 — 교수 언급

무정부주의 원래 의미 = 가장 평화적·목가적 (농촌에서 자족·공동생활)
1960년대 히피들에게 큰 영향 → 꽃 꽂고 자족 공동체 생활, 국가·엘리트·문화 권력으로부터 자유
히피의 자유 = 프루동적 의미 (직접 일하는 사람의 자유, 국가 권력으로부터의 자유)

03 자본의 시대 (1848~1870년) — 시대 개관
1848년 2월 혁명
혁명 개요

프랑스에서 시작 → 전 유럽으로 확산 (1789년 이후 최대 혁명)
주도 세력: 급진적 (몰락) 수공업자
결과: 궁극적 실패 — 총선 급진파 패배, 나폴레옹 3세 대통령 당선 → 황제 선언
전유럽: 이른바 '반혁명'의 물결

역설적 결과 — '분배' 개념의 성장

혁명에 놀란 자본가 → 노동자 달래기의 필요성 깨달음
→ '분배' 개념 성장 (오언식 온정주의 확산의 계기)

1850년대 대호황 & 사회주의 인기 하락
현상내용영향
철도 건설 붐 초기 인프라 투자 → 취업·소비 증가 → 대호황
(투자 초기는 항상 호황 — 경제 법칙)		 자본주의 대세론 확산
보수주의 변신 지주들이 토지 빨리 처분·재투자가 유리함을 깨달음
→ 자본주의 친화적 보수주의로 변신		 현대 보수정당의 원형 형성
노동 귀족 등장 온정주의 혜택 흡수한 노동자 → 중간계급 가치 수용
(근면·프라이버시·커리어·시간 엄수)		 사회주의 투쟁성 약화
결과: 사회주의 인기 치명적 하락. 이 시기에 마르크스·엥겔스가 등장해 새로운 사회주의 이론을 체계화함.

05 마르크스주의 사회주의 사상 — 4범주
4범주 전체 개요

① 인간관

1848 이전 저작 중심
경제철학 수고, 독일이데올로기

  • 헤겔 영향: Ausbildung·자기실현
  • 노동 = 자기규정 실현 도구
  • 산업자본주의에서 노동의 소외

② 역사관

역사유물론 = 마르크스주의 가장 중요한 정체성

  • 토대(생산양식) → 상부구조 결정
  • 역사 5단계 발전 법칙
  • 계급투쟁 → 사회혁명

③ 경제관

자본Ⅰ 중심

  • 자본주의 = 착취를 "은폐"
  • 잉여가치론
  • 노동자 = 저평가된 역사 발전 주체

④ 위기론

자본Ⅱ·Ⅲ, 반뒤링론, 유토피아와 과학

  • 수요론: 과잉생산·과소소비
  • 생산론: 이윤율 저하 경향
  • 위기의 주기성 설명
① 인간관 — 노동의 소외
헤겔의 영향 — Ausbildung (자기계발·자기실현)

인간 = 노동을 통해 자기규정(Ausbildung)을 실현하는 존재
"역사란 인간이 자연 상태로부터 노동을 통해 자기규정의 실현을 하면서 발전해 온 것"

산업자본주의의 문제 — 노동의 소외 (Alienation)

프롤레타리아의 노동은 자아실현의 수단이 되지 못하고 外化(alienation)한다
→ 적대적인 세계를 창출 = 노동의 소외
"노동자는 노동하지 않을 때 비로소 편안" = 노동과 여가의 이분법

초기 저작의 의미

1930년대 소련의 마르크스-엥겔스 연구소에서 발굴
1960년대 사회운동·신좌파에 큰 영향 → 자본주의 비판 + 소련식 현실사회주의 비판에 동시에 활용됨

② 역사관 — 역사유물론 (Historical Materialism)
✦ 역사유물론 핵심 명제 — 슬라이드 원문 (외워!)

"The material mode of production conditions(bedingt) the social, political, and intellectual/spiritual dimensions of life."
= 물질적 생산 방식이 사회적·정치적·사상적·정신적 삶을 조건 짓는다

물적 조건의 두 종류

① 지리·인구 구조·음식 등
② 인간 간 관계를 전제하는 경제 체제 = 생산양식 (가장 중요)
→ 생산양식 = 토대(Foundation) / 나머지 비물적 현상 = 상부구조(Super-structure)

생산양식 변화 메커니즘

마르크스·엥겔스의 기여 — "역사와 사랑에 빠짐"

기존 사회주의자들도 미래를 얘기했지만 상당수 복고적·도덕적
마르크스·엥겔스: 계몽주의의 '진보'를 사회주의와 결합 → 진보의 법칙화·체계화
→ 사회주의로의 이행이 역사 법칙상 필연이라는 주장 = 마르크스주의의 결정적 특징

③ 경제관 — 잉여가치론
자본주의와 착취 은폐

과거 생산양식(노예제·봉건제): 착취가 눈에 보임 (강제 노동 등)
자본주의: 착취를 "은폐" — 자유로운 계약처럼 보이지만 사실은 착취

단계내용
① 상품화 자본주의 하 만물의 상품화 (노동력도 상품)
② 교환가치 재화 간 거래: 동일한 교환가치를 가진 것끼리 교환
③ 노동가치설 '사회적으로 필요한 노동시간' = 추상 노동이 가치 결정 (리카도에서 수용)
④ 자본가 입장 설비+노동력을 교환가치대로 구입 → 생산 → 판매
유통 영역에서 가치 증식 없음 → 이윤은 어디서?
⑤ 잉여가치 — 핵심 생산 영역: 노동력의 특수성
'노동의 가치'와 '노동력의 가치' 사이의 모순

📌 프루동 — 심리적·경제적 근거 논박

T
프루동은 소유권의 심리적 근거(보편적 동의)가 평등에서 출발해 불평등으로 귀결한다는 점에서 논리적 모순이라고 비판했다.
→ 8차시 핵심 포인트. 로크 논리 = 평등한 주체 전제 → 결론 = 불평등 옹호 = 모순.
T
프루동의 경제적 근거 논박 중 '사회적 논박'은 노동에는 기존 사회의 지식·도구가 이미 결부되어 있다는 것이다.
→ 정확. 혼자 한 것 아님 = 사회적 기여 → 소유권 주장 불가.
F
프루동의 경제적 근거 논박 중 '물적 논박'은 노동의 결과물이 너무 많다는 것이다.
→ 새로 창출한 것이 아니라 주어진 자연을 변형했을 뿐이므로 소유 근거가 없다는 것.

📌 부조주의(Mutualism) 관련

F
프루동은 대규모 생산수단을 국가가 소유해야 한다고 주장했다.
→ 조합이 소유. 국유화는 중앙 관료제 = 노예제라고 반대. 국가 없는 조합 소유 지향.
F
프루동과 공화주의자는 소규모 생산수단에 대해 동일한 입장을 취한다.
→ 핵심 차이. 공화주의 = 소유권 인정 / 프루동 = 소유권 불허, 용익권만 인정.
T
프루동의 부조주의에서 사회 구조는 Free Commune → 지역 Commune → Confederation 순으로 아래에서 위로 구성된다.
→ 탈중앙집중화의 핵심 구조.
T
프루동은 무정부주의적 사회주의의 시초격으로, 초기 무정부주의는 가장 평화적·목가적인 이념이었다.
→ 정확. 이후 테러주의로 변모하는 것은 나중 일. 원래 성격은 자족적·목가적.

📌 자본의 시대·마르크스·엥겔스 생애

T
1848년 2월 혁명은 급진적 몰락 수공업자들이 주도했으며 궁극적으로 실패했다.
→ 정확. 총선 급진파 패배, 나폴레옹 3세 당선 → 황제 선언.
F
1850년대 대호황으로 사회주의의 인기가 크게 올랐다.
→ 반대. 호황 → 자본주의 대세론 + 노동 귀족 등장 → 사회주의 인기 치명적 하락.
T
마르크스와 엥겔스가 본격 협업을 시작한 계기는 1844년 8월 파리 Café de la Régence에서의 재회다.
→ 정확. 이 자리가 두 사람 협업의 공식적 시작점.
F
공산주의 선언은 의인동맹이 결성될 때 작성된 선언문이다.
→ 의인동맹이 공산주의 연맹으로 개칭(1847)할 때 작성된 개칭 선언문.

📌 마르크스주의 4범주

T
역사유물론은 마르크스주의의 가장 중요한 정체성이며, 생산양식(토대)이 상부구조를 조건 짓는다는 이론이다.
→ 슬라이드 원문 그대로. 반드시 외워야 함.
T
마르크스의 역사 발전 단계는 원시공산제 → 노예제 → 봉건제 → 자본주의 → 사회주의 순이다.
→ 반드시 외워야 할 5단계.
F
마르크스의 노동의 소외론에 따르면, 자본주의에서 노동자는 노동할 때 가장 자유롭다고 느낀다.
→ 정반대. "노동자는 노동하지 않을 때 비로소 편안" = 노동과 여가의 이분법.
F
마르크스의 잉여가치는 유통 영역에서 발생한다.
→ 생산 영역에서 발생. "유통영역의 가치 증식 없음" = 슬라이드 핵심 포인트.
T
마르크스의 위기론에서 수요론은 룩셈부르크가 훗날 완성했고, 사회화된 생산과 생산물의 사적 소유 사이의 모순이 핵심이다.
→ 슬라이드 원문 내용.
F
마르크스·엥겔스는 기존 사회주의자들처럼 도덕적 근거에서 사회주의의 필요성을 주장했다.
→ 결정적 차이. 마르크스·엥겔스 = 역사의 법칙으로서 사회주의 이행의 '필연성' 주장. 도덕 아닌 과학적·역사적 근거.

✦ 논술형 대비 핵심 논지 (8차시)

  • 프루동의 사유재산권 3가지 근거 논박을 모두 서술하고, 이 중 경제적 근거 논박이 가장 강력한 이유를 설명하라.
  • 프루동의 부조주의가 공산주의와 어떻게 다른지, "관리받는 공동체원보다 자유로운 조합원"이라는 표현을 중심으로 서술하라.
  • 프루동과 공화주의가 소규모 생산수단 소유권에서 어떻게 다른지 서술하라.
  • 1848년 혁명 실패 이후 자본의 시대(1848~1870)에 사회주의 인기가 하락한 이유를 노동 귀족 등장과 연결해 서술하라.
  • 역사유물론의 핵심 명제를 영문 원문과 함께 설명하고, 역사 5단계 발전 메커니즘을 서술하라.
  • 마르크스의 잉여가치론이 왜 자본주의의 착취를 "은폐"라고 보는지 설명하라.
  • 마르크스·엥겔스가 기존 사회주의자들과 결정적으로 다른 점이 무엇인지 (도덕 vs 역사 법칙) 서술하라.
