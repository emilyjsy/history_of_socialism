01 역사란 무엇인가
역사의 정의 — 3단계
핵심 정의

역사 = 특정 과거(the past)를 주관적으로 재현한 하나의 이야기(an account)
단, ① 합리적(rational)이어야 하고 ② 맥락 중심적(contextual)이어야 함

구분 의미 포인트
the past 그냥 "과거 일반"이 아닌 특정 과거 관사 'the' — 특정성
an account 여럿 중 하나의 이야기 (account = 이야기, 설명) 관사 'an' — 복수 가능성, 주관성
represent 전체를 다 드러낼 수 없어 부분만 재현 (선택과 집중) present ≠ represent
교수 예시 — 서울대학교 사진

서울대를 사진 하나에 담으라 → 샤 찍는 사람 / 총장 잔디 찍는 사람 / 도서관 찍는 사람 각각 다름. 선택과 집중이 곧 representation. 나머지는 다 포기한 것.

교수 예시 — 한국사 교과서

"한민족이 반만 년 동안 하나" → 민족 통합 관점에서 쓴 하나의 account. 호남·영남이 달랐다는 관점으로 전혀 다르게 쓸 수도 있음. 국가 유지를 위한 특정 관점의 account에 불과.

교수 예시 — 6.25 전쟁

6.25도 특정한 the past. 그런데 구현 방식은 군사 서술 / 민간인 고통 / 지역별 경험 등 다양. 교과서의 "새벽 남침" 서술은 그 중 하나의 방식.

역사의 2가지 조건
조건 ①

합리성 (Rationality) — 논쟁 가능한 수준의 합리성 경계 안에 있어야 함

교수 예시
  • ❌ "6.25는 천지신명의 뜻이었다" → 합리적 설명 불가
  • ❌ "일제 강점기는 일본 천왕 폐하의 은덕" → 합리적 설명 불가
  • ✅ "일제 강점기가 경제적으로 어느 정도 도움이 됐다" → 경제 지표로 설명 가능, 논쟁 가능
조건 ② (핵심!)

맥락 중심성 (Contextuality) — 사건·말·개념만이 아니라 그것이 발생한 상황·배경을 함께 고려

교수 예시 — 전해 들은 말

친구한테 전해 들은 말이 기분 나쁜 이유? → 맥락(context)이 없기 때문. 실제로는 농담 분위기, 술자리 등 상황(context)이 있었다면 전혀 다른 의미. 맥락 = 텍스트를 둘러싸고 있는 상황·배경.

교수 예시 — 물이 끓는 온도

물은 "100도에 끓는다"가 법칙. 그러나 일본 후지산 중턱은 94.2도, 중국 강가는 다름. 맥락이 들어올수록 그때그때 다 다르다가 됨 → 법칙 추출 불가 → 과학성 낮아짐. 역사는 과학성보다 맥락(당대 진실)에 충실.

⚠️ 시험 포인트

역사학은 "과거를 통해 교훈을 얻는 학문"이 아니다.
역사학 = "그땐 그랬지"를 맥락과 함께 드러내는 것.

역사의 효용 — "현재를 낯설게 하기"
역사의 진짜 효용

과거를 맥락 중심으로 파고들면 → 현재의 제도·관행이 당연한 것이 아님을을 깨달음
→ 현재의 권위·정당성에 도전하거나 비판할 근거가 됨

과거 집중 탐구

교수 예시 — 외국 생활 비유

외국에 오래 살다 오면 한국의 불합리한 관행이 갑자기 눈에 보임 → 낯선 곳(=과거)을 경험하면 현재가 낯설어짐.
영국 작가 표현: "역사는 또 하나의 외국이다"

교수 예시 — 대기업 불합리한 제도

취직 후 불합리한 제도를 보면 어른들은 "원래 그런 거다"(과거의 무게)로 정당화. 그러나 역사 공부한 사람은 그 제도가 특정 맥락 속에서 우연히 만들어진 것임을 알 수 있음 → 강력한 비판 무기.

수업 목표

자본주의·시장경제를 "당연한 것·인간 본성"으로 보지 않고 → 200년 전 특정 맥락에서 형성된 하나의 제도로 낯설게 보는 것. 사회주의 역사를 통해 그 도전들을 살펴봄.

02 산업화 (Industrialization)
산업화의 정의
정의

생산의 동력(에너지원)이 생체 근력(인간+동물) → 비(非) 생체 동력으로 변화한 현상
= 공장제 기계 공업

구분 에너지원 특징
산업화 이전 생체 근력 (인간 + 동물) 쉬어야 함, 한계 있음
초기 산업화 증기기관 (물 고열로 끓임 → 에너지) 안 쉬어도 됨, 압도적 출력
이후 전력, 내연기관(석유), 원자력, 신재생에너지 다양화
교수 예시 — 방적기

초창기 방적기는 사람이 네 발로 돌림 → 힘듦. 증기기관 접목 → 기계는 안 쉬어도 됨. 특정 공간(공장)에서 다수의 인간 + 기계로 대량 생산 가능.

초기 산업화의 대표 모습 2가지

① 면방직(면공업) 공장    ② 철도(증기 기관차)

시험 포인트: 산업화 = 공장제 기계 공업. 동력 변화가 핵심. 산업혁명의 기원은 18세기 중엽 영국이지만, 유럽 전체에 유의미한 영향력을 발휘한 건 19세기 넘어서 (벨기에 19세기 초, 프랑스 1850년경).
03 자본주의 (Capitalism)
자본주의의 정의 — 아카데믹 2가지
정의 ① — 유통 형식으로서의 자본주의

재화가 시장에서 화폐를 통해 유통되는 시스템
= 시장 경제
= 만물의 상품화 (마르크스 표현, 다른 학자들도 사용)

만물의 상품화 의미

지금 갖고 있는 물건 중 내가 직접 만든 것? → 거의 없음. 모든 재화를 시장에서 구매. 시장 판매를 위해 생산. 이것이 자본주의 이전에는 당연하지 않았음.

핵심: 자본주의 이전에는 상품이 아니었던 것들
자본주의 이전 → 상품화되지 않았던 2가지 + 생산수단

① 노동 (Labor)
② 토지 (Land) — 생산 수단 대표

자본주의 이전

노동·토지는 시장 거래 대상 아님. 생계 = 내 땅에서 직접 생산하거나 비시장적 방식.

자본주의 이후

노동·토지 모두 상품화. 취업 = 노동력 판매. 부동산 = 토지 상품화.

교수 예시 — 노동의 상품화 (취업)

취업 = 내 노동력을 고용주에게 파는 것. 고용주 = 노동력을 산 것.
일시불 아닌 월급 형태인 이유 → 계속해서 노동 제공을 증명해야 하기 때문.
태블릿 구매 비유: 고용주가 나를 인간적으로 좋아서가 아니라, 태블릿 사듯 역량을 구입하는 것.

교수 예시 — 토지의 상품화

오늘날 토지 = 가장 먼저 "땅값" 떠올림 → 이미 상품. 자본주의 이전에는 시장 거래 대상이 아니었음. 자본주의 이전: 복수의 사람들 생계가 걸린 것 → 일반 상품 취급 안 함.

생산 수단 (Means of Production) 4가지
생산 수단 생산물 특징
토지 농산물 등 복수의 생계 연결
임야 목재, 과일 등 복수의 생계 연결
광산 금, 석탄 등 지하자원 복수의 생계 연결
공장 제조품 복수의 생계 연결
공통 특징: 복수의 사람들의 생계가 걸려 있음 → 자본주의 이전에는 일반 상품으로 취급하지 않음.
※ 바다는 예외 (소유·상품 개념 없음)
04 사회주의 등장 배경
사회주의(Socialism) 용어 기원
등장 시기

오늘날 우리가 쓰는 의미의 "사회주의" 용례 → 약 200년 전 (19세기 초~중반)
오언(Owen)의 제자들이 처음 사용.
한국 기준: 세도정치, 안동 김씨, 영정조 이후 시기에 해당.

오해 주의: 고대 중국 균전제, 정약용의 여전제, 망이망소이의 난 같은 토지 분배 운동은 사회주의와 관련 없음. 사회주의는 산업 자본주의 이후의 개념.
사회주의 등장의 전제: 산업 자본주의
명제

산업 자본주의가 없으면 사회주의도 없다.
사회주의 = 산업 자본주의라는 배경 아래에서 탄생한 이념.

✦ 수업의 목표

  • 자본주의·시장경제를 당연한 인간 본성이 아닌, 특정 맥락에서 만들어진 제도로 낯설게 보기
  • 사회주의 역사를 통해 자본주의에 대한 도전의 역사 이해
  • "사회주의가 더 좋다"는 결론을 내리는 것이 목적이 아님 (교수 명시)

교수 예시 — 근대 경제학의 전제

"인간의 욕망은 무한하지만 재화는 유한하다" → 경제학 원론 첫 페이지.
그러나 역사를 보면: 인간의 욕망이 항상 무한했나? 언제부터 무한했나?
→ 역사를 가보면 꼭 그렇지 않음. 이 전제 자체도 특정 시대의 산물.

05 T/F 예상 포인트 — 까다로운 함정 정리

📌 역사 관련

F
역사는 과거 그 자체다.
→ 과거에 대한 하나의 account(이야기). 과거 자체와 역사는 다름.
F
역사는 the account이다. (즉 유일한 하나의 설명이다)
→ an account. 관사 'an' → 여럿 중 하나. 'the'가 붙으면 유일한 설명 → 틀림.
F
역사는 과거를 통해 교훈을 얻는 학문이다.
→ 교수가 명시적으로 부정. 맥락이 달라 적용 불가. 역사는 "그땐 그랬지"를 드러내는 것.
F
역사학은 과학성이 높은 학문이다.
→ 맥락 중심 → 법칙 추출 불가 → 과학성 낮음. 교수 언급: 인문사회 학문 중 거의 유일하게 맥락 중심.
T
역사는 특정 과거(the past)를 주관적으로 재현한 하나의(an) account이다.
→ 정확한 정의. 관사 포함 외워야 함.
T
역사는 현재 제도의 정당성·권위에 도전하는 무기가 된다.
→ 역사의 효용 = 현재를 낯설게 해서 권위를 무너뜨림.
F
한국사 교과서는 유일하게 올바른 한국 역사 서술이다.
→ 민족 통합 관점에서 쓴 하나의 account일 뿐. 호남·영남이 달랐다는 관점 등 다른 서술도 가능.
F
나당 전쟁과 오늘날 한중 관계는 역사적으로 비교 가능하다.
→ 맥락이 완전히 다름. 역사학에서는 맥락이 다른 시대 간 단순 비교 불가.

📌 산업화 관련

F
산업화와 자본주의는 같은 말이다.
→ 다른 개념. 산업화 = 동력 변화, 자본주의 = 유통 형식. 둘이 결합한 것이 산업 자본주의.
T
산업화 이전 생산의 주 동력은 생체 근력(인간+동물)이었다.
→ 정확한 표현. '생체 근력' = 인간 + 동물.
F
산업화는 18세기 중엽부터 이미 유럽 전체에 본격적으로 퍼졌다.
→ 영국에서 시작됐으나 유럽 전체 확산은 19세기. 프랑스는 1850년경.
T
초기 산업화를 대표하는 것은 면공업(면방직)과 철도이다.
→ 정확. 두 가지 기억.

📌 자본주의 관련

T
자본주의를 "만물의 상품화"로 표현할 수 있다.
→ 마르크스 등이 사용한 표현. 유통 형식으로서의 자본주의 정의.
F
토지는 자본주의 이전에도 시장에서 거래되는 상품이었다.
→ 자본주의 이전에는 상품 아님. 복수의 생계가 걸려 일반 상품 취급 안 함.
T
자본주의에서 노동은 상품화되어 있다. 취업은 노동력을 파는 행위다.
→ 노동시장, 월급 = 노동력 판매의 구조.
F
자본주의 이전에도 생산 수단은 일반 상품처럼 거래됐다.
→ 아님. 복수 생계 연결 → 시장 거래 대상 아니었음.

📌 사회주의 관련

F
정약용의 여전제나 중국 균전제는 사회주의의 초기 형태이다.
