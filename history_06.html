01 로치데일 조합 — 최초의 현대적 노동조합
로치데일(Rochdale) 조합이란
정의

맨체스터 인근 로치데일에서 탄생한 조합
= 현대적 노동조합의 시초로 불림
오언 조합보다 훨씬 온건(moderate)하고 현실적

구분오언 GNCTU로치데일 조합
생산수단 소유 조합이 소유 지향 (사회주의적) 건드리지 않음 — 사장 재산은 사장 것
주요 활동 생산수단 조합 소유 추구 공동구매(공구)·임금 협상 중심
사회주의 여부 사회주의적 사회주의 운동으로 볼 수 없음 (소유권 건드리지 않음)
현대 연결 현대 노조와 다름 현대 노동조합(민노총 등)의 직접 조상
핵심 포인트

로치데일 = 현대 노동조합 시초 BUT 생산수단 소유권 문제 건드리지 않음
→ 사회주의 정의(생산수단의 사회적 소유)에 해당하지 않음
→ 중간고사 이후 배울 개혁주의·개량주의 사회주의와 유사한 성격

교수 예시 — 현대 노동조합과 비교

현대 민노총: "울산공장 일부가 우리 겁니다"가 아님 → 임금·노동조건 협상이 핵심
로치데일도 마찬가지 → 소유권 ❌, 공동구매·임금 ✅
(교수: 민노총이 국제 기준에서는 사실 굉장히 온건한 편)

02 에띠엔 까베 (Étienne Cabet) & 이카리아 여행기
까베 생애 & 등장 배경
항목내용
출신수공업자 집안. 집안은 몰락했지만 본인은 능력자 → 변호사로 성공
사상 계기산업자본주의의 불공정함을 느끼며 민중 잡지 르 포퓰레르(Le Populaire) 발행
망명왕정 탄압으로 1834년 영국 망명 (1834~1839년)
영국 체험산업화된 공장·도시 직접 목격 → 더욱 확신 → 나머지 인생을 사회 변혁에 헌신 결심
대표 저작망명 중 집필한 이카리아 여행기 (사회주의 역사의 고전)
귀국 후1839년 파리 귀환. 르 포퓰레르 재발행 → 구독자 40만 명 → 프랑스 지식인계 최고 아이콘
이카리아 여행기 — 핵심 내용
이카리아 여행기 개요

가상의 섬나라 이카리아를 통치하는 이카르 (예수 이미지의 인물) 가 다스리는 이상향을 여행기 형식으로 묘사
→ 철학적 논문이 아닌 여행기 형식 → 대중이 접근하기 쉬움 → 큰 반향
→ 까베의 추종자들 = 오언주의자 아닌 "이카리아인"으로 자처

① 경제

  • 사유재산 전면 폐지 (생산수단만이 아닌 모든 재산)
  • 완전 국유화
  • 시장 메커니즘 없음 → 중앙계획으로 분배
  • 국가산업위원회가 생산량 목표·분배 결정
  • 화폐 소멸 (내부 거래 시) — 외화만 존재
  • 각 공장에 민주적 회의체 (관료주의 방지)
  • 동기 부여: 유용한 경쟁 = 표창장·영웅 칭호 (시장 없어도 명예 경쟁 있음)

② 정치

  • 2,000명 의회 (입법 + 행정 통합)
  • 의회 내 위원회가 행정 역할 (별도 정부 없음)
  • 명칭: 인민대표회의
  • 사법: 인민재판 = 법률 엘리트가 아닌 인민이 참여해 결정 (배심원제 확대 형태)
  • 인민재판의 핵심: 법정 절차 간소화 + 민주성

③ 문화

  • 출판사 하나 (국가 소유)
  • 신문 하나 (국가 승인, 인민 선출 대표가 편집)
  • "이론(異論) 없음" — 이유: 생산수단 소유권 갈등이 없으면 근본적 견해 차이 없음 (까베 논리)
  • 개인 취향 차이는 인정. 사회·경제적 계급 갈등에서 오는 근본 이론만 없음

④ 일상

  • 6시 기상 → 간단한 아침 → 9시 제대로 아침
  • 2시 점심: 거대 공동 식당에서 모든 이카리아인 함께 (집단의식 함양)
  • 4시: 국가가 집 앞에 식재료·음식 배급
  • 5~6시: 자유롭게 저녁 식사
  • 슬로건: "각자에게는 그 필요에 따라, 각자로부터는 그 능력에 따라"
✦ 이카리아 핵심 슬로건 — 외워!

"각자에게는 그 필요에 따라, 각자로부터는 그 능력에 따라"
= 능력껏 기여하고, 필요한 만큼 가져가라. 기여 많다고 더 가져가지 않고, 기여 적다고 쫄지 않는다.

까베 vs 오언 비교 — 시험 포인트

오언 협동촌락: 소규모 공동체, 생산수단 소유권 미언급, 자본가 투자 기반
까베 이카리아: 국가 규모, 모든 사유재산 폐지, 중앙계획 경제, 화폐 소멸
→ 까베가 훨씬 더 국가주의적이고 급진적

03 공산주의(Communism) 용어 탄생 — 1840년대
왜 '공산주의'라는 새 용어가 나왔나
공산주의 용어 등장 배경 (1840년~)

당시 주류 사회주의 = 오언주의자
까베 계열: 오언식 사회주의와 같은 대의 공유하지만
방법·궁극적 목표에서 차별화 → "우리는 더 진짜다"
→ communist 용어 채택. 영국의 바름비(Barmby)가 영어권에 유행시킴

구분사회주의 (오언 계열)공산주의 (까베 계열)
공통점생산수단 사적 소유 반대 / 시장 경제에 제3자 개입 필요
재산생산수단 소유권 문제 중심모든 사유재산 폐지 (더 급진)
규모공동체 단위국가 단위 (이카리아 = 섬나라 국가)
경제 운영다양 (협동촌락·협동기업 등)중앙계획 + 화폐 소멸
성격다양한 방식기독교적 + 국가·재산공유공동체
공산주의 용어의 역사적 뿌리
재세례파 (Anabaptists) — 공산주의 이미지의 원형

16세기 종교개혁 시기 독일 뮌스터에서 활동한 급진 신교 공동체
자기 재산을 다 공동체에 헌납하고 완전한 평등 공동체 생활
→ 이들을 "공산주의자"라고 불렀던 역사적 이미지가 남아 있었음
→ 까베 이카리아 구상과 유사 → 용어 자연스럽게 부활

기독교와 공산주의의 연결 — 시험 포인트

예수의 가르침 = 자발적 가난 + 평등 공동체 (기독교 본래 성격)
중세 수도원 농사·프란치스코 탁발승 = 자발적 가난의 연장
재세례파 → 까베 이카리아 → 초기 공산주의
→ 초기 공산주의 = 다분히 기독교적 성격
→ 이후 마르크스가 이걸 세속화·과학화 (큰 변화)

아이러니: 공산주의 용어는 기독교적 배경에서 까베가 퍼뜨렸지만, 얼마 안 가 블랑키와 마르크스 등에 의해 전혀 다른 의미를 갖게 됨. 까베가 용어의 주인공 자리를 금방 빼앗김.
04 루이 오귀스트 블랑키 (Louis Auguste Blanqui) — 등장
블랑키 생애 & 배경
항목내용
출신프랑스 남부 유지의 아들. 아버지 = 관료. 형 = 당시 프랑스 손꼽히는 경제학자
학력파리대학 법학부 입학(서울 법대 수준) + 의학부 복수 전공. 초엘리트 가도
전환점1820년대 초반(20대 초반) 급진 공화주의자들의 글을 읽기 시작 → 삐딱선
사상 영향프랑스 혁명 급진파 바베프(Babeuf)의 생애와 사상 → 쿠데타형 혁명 발상 형성
7월 혁명 (1830)샤를 10세 쫓아냄 → 열광 → 그러나 경제 시스템 그대로 → "혁명이 왜 이것밖에 안 해?" 분노
선택기존 사회주의 방향 아닌 새로운 옵션: 정치적 쿠데타·봉기 방식 → 다음 차시 상세 학습
공화주의 — 사회주의와 차이
공화주의 (블랑키가 초기에 접한 사상)

반군주정 + 소생산자들의 세상
소생산자 = 소규모 수공업자·자영농
→ 이들이 각자의 작은 생산수단 소유권을 확보하며 사는 세상 지향

구분공화주의사회주의
생산수단 소유소규모 소유는 긍정 (자영농·소장인의 소유권 확보)소유권 자체를 문제화
길드길드 개입 싫어함 (개인 자유 우선)길드의 공동체 역할 일부 긍정
비판 대상대형 공장주·대지주만 문제생산수단 사적 소유 전반 문제
이상 사회뛰어난 장인이 자기 가게 갖고 존경받으며 사는 사회생산수단 사회적 소유·운영
블랑키의 선택

7월 혁명 실망 → 공화주의 수준으로는 부족 → 사회주의자 선언
그러나 생시몽·오언·푸리에·까베 방식도 아닌 새로운 옵션:
바베프의 쿠데타 시도에서 영감 → 정치적 봉기·혁명 방식
→ 상세한 내용은 다음 차시

바베프 (Babeuf) — 블랑키에 영향을 준 인물
프랑수아 바베프 (François Babeuf)

프랑스 혁명 당시 가장 급진적인 쿠데타(봉기) 시도를 했던 인물
봉기 직후 체포·처형됨
→ 블랑키가 바베프의 친구를 스승으로 삼으며 혁명적 방법론에 영향받음

05 T/F 예상 포인트 — 6차시 함정 정리

📌 시기·맥락 관련

F
로치데일·까베·블랑키 등의 활동은 1830년 이전의 일이다.
→ 모두 1830년 이후. 교수가 직접 강조한 시험 포인트. 시기 혼동 = 틀림.

📌 로치데일 조합 관련

F
로치데일 조합은 생산수단의 조합적 소유를 목표로 했다.
→ 생산수단 소유권 건드리지 않음. 그것은 오언 GNCTU. 로치데일은 공동구매(공구)·임금 협상 중심.
T
로치데일 조합은 현대적 노동조합의 시초로 불리지만, 사회주의 정의에 해당하지 않는다.
→ 생산수단 소유권 불건드림 = 사회주의 정의 미충족.

📌 까베 & 이카리아 관련

T
까베는 이카리아 여행기를 영국 망명 중에 집필했다.
→ 1834~1839년 영국 망명 중 집필.
F
이카리아는 생산수단만 사회화하고 개인 사유재산은 허용한다.
→ 모든 사유재산 전면 폐지. 오언식(생산수단만 문제화)과 다른 까베의 특징.
T
이카리아에는 시장 메커니즘이 없고 국가산업위원회의 중앙계획으로 분배한다.
→ 정확. 화폐도 내부적으로 소멸 (외화만 존재).
F
이카리아에는 경쟁이 전혀 없다.
→ 시장 경쟁은 없지만 "유용한 경쟁"(표창장·영웅 칭호)은 존재. 까베가 직접 제시한 동기 부여 방안.
F
이카리아에는 출판의 자유가 있고 다양한 출판사가 존재한다.
→ 출판사 하나(국가 소유), 신문 하나(국가 승인). 이론(異論) 없음 = 계급 갈등이 없으면 근본 견해 차이 없다는 논리.
T
이카리아의 핵심 슬로건은 "각자에게는 그 필요에 따라, 각자로부터는 그 능력에 따라"이다.
→ 반드시 외워야 할 슬로건.
T
이카리아의 인민재판은 법률 엘리트 대신 인민이 참여하는 배심원제 확대 형태다.
→ 까베의 인민재판 정의. 절차 간소화 + 민주성 강조.
F
까베의 르 포퓰레르는 귀국 후 약 4만 명의 구독자를 모았다.
→ 40만 명. 당시 기준 어마어마한 숫자.

📌 공산주의 용어·블랑키 관련

T
공산주의(communist)라는 용어는 1840년부터 본격적으로 쓰이기 시작했다.
→ 정확. 까베 계열이 사용, 영국의 바름비가 영어권에 유행시킴.
F
공산주의는 처음부터 마르크스의 무신론적·세속적 사상이었다.
→ 초기 공산주의는 다분히 기독교적 성격. 재세례파 이미지에서 유래. 마르크스가 이후 세속화·과학화함.
F
공화주의는 모든 생산수단의 사회적 소유를 지향하는 사회주의와 같은 이념이다.
→ 다름. 공화주의 = 소규모 생산수단 소유 긍정 + 반군주정. 사회주의 = 생산수단 소유권 자체를 문제화.
T
블랑키는 7월 혁명(1830)에서 경제 시스템이 그대로 유지되는 것을 보고 기존 사회주의 방식 대신 정치적 봉기 방식을 선택했다.
→ 블랑키의 전환점. 상세 내용은 다음 차시.
F
블랑키는 영국 출신으로 산업화의 폐해를 직접 경험한 인물이다.
→ 프랑스 출신. 프랑스 남부 유지의 아들, 파리대학 법학·의학 복수 전공 엘리트.

✦ 논술형 대비 핵심 논지 (6차시)

  • 로치데일 조합이 현대 노동조합의 시초임에도 사회주의 운동으로 볼 수 없는 이유를 서술하라.
  • 까베의 이카리아 여행기가 오언의 협동촌락 구상과 어떻게 다른지, 국가주의적 성격을 중심으로 비교하라.
  • 이카리아의 경제 운영 원리(중앙계획·화폐 소멸·유용한 경쟁)를 설명하고, 까베가 어떻게 동기 부여 문제를 해결하려 했는지 서술하라.
  • '공산주의'라는 용어가 1840년에 등장한 맥락과, 초기 공산주의의 기독교적 성격을 재세례파와 연결해 설명하라.
  • 공화주의와 사회주의가 생산수단 소유권에 대해 어떻게 다른 입장을 갖는지 비교하라.
